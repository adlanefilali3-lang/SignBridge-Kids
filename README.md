## About the Project

SignBridge Kids is an educational and assistive technology project designed to help children learn and translate sign language using a smart glove based on the BBC micro:bit.

The smart glove detects finger movements and hand orientation, then sends the data wirelessly to a mobile application via Bluetooth. The application translates sign language into text and speech while also providing an interactive learning mode for children.

The project aims to make communication easier between deaf and hearing people and promote inclusive education.

---

## Objectives

- Translate sign language into text.
- Convert translated text into speech.
- Help children learn sign language through interactive activities.
- Promote communication between deaf and hearing communities.
- Develop an affordable educational solution using micro:bit.
